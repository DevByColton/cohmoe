// PIRATE SOFTWARE GAME JAM

Maestro was created in 2 weeks during a pirate software game jam using Godot, written in GDScript. The theme for the game jam was "You are the weapon". So a friend and I thought it would be fun to make a rhythm game where you are the music notes fighting back against an evil maestro.

We chose Godot for this game jam because we needed to just make the game and not worry about the engine side. Plus, we needed the game to be able to play in a browser for upload to itch.io. For that reason you can actually play the web build of Maestro here on my site. There is a song that plays along to the gameplay, so sound up!

The gameplay is nothing fancy. Use A and D move left and right, space to jump, and QWERT to hit the notes as you come into contact with them. This game jam was good experience for me because of the time limit. We only had two weeks to come up with the game idea once we got the prompt, create all of the assets, and code it.
